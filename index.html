Antes de empezar vamos a repasar Git y GitHub

Crear un nuevo repositorio en Git

    Git Init
  • Me ubico en la carpeta donde quiero crear el proyecto: cd
  • Inicio el proyecto con el nombre que le asigné al repositorio "JavaScript": git init javascript
  • Comprobamos que el repositorio se creó: dir javascript
  • Me ubico en el repositorio: cd javascript
      • Git Status
  • Con Git Status chequeamos el estado del repositorio: git status
      • Git Add
  • Se usa Git Add para adherir el archivo para subirlo al repo
  • Por ejemplo: git add index.html

    • Una vez que los archivos estan en el Staging Area hay que pasarlos al repositorio local con Git Commit.

    Git commit -m "Comentario de que se trata el commit que estamos haciendo".

  • Por ejemplo: git commit -m "Repasando Git"

    • RAMAS

    Cuando queres agregar una nueva función o solucionar un error generas una rama para agregar estos cambios

    Como crear una rama?

  • git branch nuevaRama
      • Pasos para crear una rama
  • Verifico en que rama estoy: git branch
  • Creo la rama: git branch nuvaRama
  • Verifico que se creó la rama: git branch -l

    • Para movernos entre ramas: git checkout nombreDeRama

    Verifico que realmente me movi: git branch -l

      Como subir el repositorio
  • Me ubico en el repositorio con "cd"
  • git remote add origin https://github.com/miuser/javascript.git
  • git push -u origin master
